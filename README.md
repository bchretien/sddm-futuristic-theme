Futuristic Theme for SDDM
---------------------------------
SDDM is a Login Manager for Linux which can be themed by qml.

Installation
------------
* Copy the folder `futuristic` to /usr/share/apps/sddm/themes/
* Change CurrentTheme to `futuristic` in /etc/sddm.conf
* Enjoy.
